# Road Pulse - Thematic Interactive Driving Game

A browser game built on Node.js and HTML5 Canvas. Pick a theme, dodge oncoming traffic, collect boost orbs, and chase your high score.

## Features

- Three visual themes: Neon City, Desert Storm, Arctic Night
- Smooth steering with keyboard and touch controls
- Progressive speed system with boost pickups
- Persistent best-score storage in browser localStorage
- Mobile-friendly layout with on-screen controls

## Run Locally

1. Install dependencies:

   npm install

2. Start the server:

   npm start

3. Open:

   http://localhost:3000

## Controls

- Left / Right arrows or A / D: steer
- Shift: activate boost when boost meter is available
- Mobile: use Left, Boost, Right buttons
